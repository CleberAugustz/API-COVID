\\ Indicadores \\

  • Infectados

    O indicador de infectados utiliza o algoritmo de contágio demonstrado por Signer. Para calcular a projeção precisamos do número de casos inicias da população análisada, no mínimo 1, a taxa de de contágio/reprodução do vírus (R). Para o COVID-19 está sendo considerada mundialmente em torno 2,5. O que significa que para uma pessoa infectada mais 2,5 pessoas serão infectadas a partir dela, esse valor é afetado diretamente pela porcentagem de isolamento social. Para um isolamento social de 50% a taxa de reprodução cai para metade 1.25 R. Com o Isolamento social com mais de 70% a taxa de reprodução do vírus fica a baixo de 1 R com isso não há novos casos de disseminação. As simulações podem ser feitas com intervalos múltiplos de 5 dias.

  • Mortos

    O indicador de número mortos usa os últimos dados divulgados pela prefeitura de Piracicaba de mortos e infectados para gerar o percentual de mortos por infectados. Utilizando esse mesmo percentual sobre o número de infectados previstos temos o número de mortos previsto.

  • Hospitalizados

    O indicador de número hospitalizados usa os últimos dados divulgados pela prefeitura de Piracicaba de hospitalizados e infectados para gerar o percentual de hospitalizados por infectados. Utilizando esse mesmo percentual sobre o número de infectados previstos temos o número de hospitalizados previsto.

  • Recuperados

    O indicador de número recuperados usa os últimos dados divulgados pela prefeitura de Piracicaba de recuperados e infectados para gerar o percentual de recuperados por infectados. Utilizando esse mesmo percentual sobre o número de infectados previstos temos o número de recuperados previsto.

  • Suspeitos

    O número de casos suspeitos não contemplam nenhuma projeção, são apenas os dados levantados pela prefeitura de Piracicaba.

  • Descartados

    O número de casos descartados não contemplam nenhuma projeção, são apenas os dados levantados pela prefeitura de Piracicaba.

  • Isolamento Social

    O percentual de isolamento social é fornecido pelo site do Estado de São Paulo sendo atualizado a cada 48 horas. Para fazer a projeção usamos a média simples dos últimos 7 dias com o objetivo de evitar distorcer a projeção por conta de valores pontuais de isolamento muito diferentes dos demais como costumam ser as taxas de isolamento social aos finais de semana.

\\ Regiões Vs Totais \\

  • Dados das Regiões

    Os dados das Regiões de Piracicaba são atualizados semanalmente diferentemente dos dados totais de Piracicaba que são atualizados diariamente. Por conta disso consideramos mais apropriado utilizar a projeção de 5 dias para os dados totáis diários e de 10 dias para os dados regionáis semanáis.

\\ Sistema de Saúde e População \\

  • Sistema de Saúde

    Todos os dados referentes ao sistema de saúde são retirados do site oficial da prefeitura de Piracicaba no portal da transparência.

  • População

    Todos os dados referentes a população de Piracicaba foram encontrados na plataforma fornecida pelo IBGE. Os dados projetados com relação a súade da população levam em consideração os percentuais levantados pela Pesquisa Nacional de Saúde e a Unifesp.
